APPENDIX A - Classes
CD Surface Location

Class Name: CD Surface Location

Superclass - <Location Surface>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies a surface coordinate for an SRF based on the Celestiodetic SRFT.

See 8.5.4 Celestiodetic SRFT of ISO/IEC 18026 for a complete definition.

Primary Page in DRM Diagram:

  • 15

Secondary Pages in DRM Diagram:

This class appears on only one page of the DRM class diagram.

Example

  1. Consider a transmittal representing Walt Disney World in Orlando, Florida. The <Transmittal Root> instance has an <Environment Root> component ER with srf_context_info specifying that no designated spatial surface is part of the SRF context info, an SRF derived from the Celestiodetic SRFT is being used with ORM SRM_ORMCOD_WGS_1984, reference transformation SRM_RTCOD_WGS_1984_IDENTITY, and specifying angular unit EUC_DEGREE_ARC and linear unit EUC_METRE. A <Feature Node> instance within the component tree of ER represents Cinderella Castle and has a <CD Surface Location> instance with a coordinate value of longitude = -81.35 and latitude = 28.25.
