Colour Table Library

Class Name: Colour Table Library

Superclass - <Library>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies a <Library> instance consisting of <Colour Table Group> components.

Primary Page in DRM Diagram:

  • 10

Secondary Pages in DRM Diagram:

  • 1
  • 14

Example

  1. Consider a data provider whose format specifies colours using palettes. To convey exactly the colour information used, each palette is represented as the primary <Colour Table> component of a <Colour Table Group> instance. The collection of these <Colour Table Group> instances is the <Colour Table Library> instance of the transmittal being produced.
