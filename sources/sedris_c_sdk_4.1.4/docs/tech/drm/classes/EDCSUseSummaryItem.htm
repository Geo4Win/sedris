EDCS Use Summary Item

Class Name: EDCS Use Summary Item

Superclass - <SEDRIS Abstract Base>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

Within the scope being summarized, an instance of this DRM class can specify either an ECC with an optional set of EACs, or just an individual EAC.

The <Classification Data> instance, which may be qualified by <Property Value> instances, represents the (potentially qualified) classification being described, if any.
