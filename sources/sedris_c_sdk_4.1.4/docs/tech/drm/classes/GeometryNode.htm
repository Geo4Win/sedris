Geometry Node

Class Name: Geometry Node

Superclass - <Geometry Topology>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class is a zero-dimensional <Geometry Topology> instance specifying the topological location of various geometry representations and/or an endpoint of one or more <Geometry Edge> instances.

The location of a <Geometry Node> instance is the same as that of the spatially located <Geometry Representation> instance or <Vertex> instance to which the node is associated. If the <Geometry Node> instance is associated to more than one such DRM object, the DRM objects shall be collocated.

Notes

Associated with Notes


Ellipse

The <Geometry Node> instance corresponds to the <Location> instance representing the centre of each <Ellipse> instance.


Geometry_Edge

This relationship indicates the collection of <Geometry Edge> instances that are bounded by the <Geometry Node> instance, if any. For each <Geometry Edge> instance associated with the <Geometry Node> instance through the use of an <Edge Direction> link object with a forwards field value of SE_TRUE, the <Geometry Node> instance is the starting node of the <Geometry Edge> instance. For each <Geometry Edge> instance associated with the <Geometry Node> instance through the use of an <Edge Direction> link object with a forwards field value of SE_FALSE, the <Geometry Node> instance is the ending node of the <Geometry Edge> instance.


Geometry_Face

This is the <Geometry Face> instance that contains the <Geometry Node> instance, if one exists.


Geometry_Volume

This is the <Geometry Volume> instance that contains the <Geometry Node> instance, if any.


Volume_Object

The <Geometry Node> instance corresponds to the <Location> instance representing the centre of each <Volume Object> instance.
