Abstract Class Name: Light Rendering Behaviour

  • <Rotating Light Behaviour>

Definition

An instance of this DRM class specifies, for the aggregating <Light Rendering Properties> instance, behaviour that is specific to a particular type of light, where type of light refers to its shape, direction, or apparent motion.
