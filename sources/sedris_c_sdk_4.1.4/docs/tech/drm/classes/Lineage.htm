APPENDIX A - Classes
Lineage

Class Name: Lineage

Superclass - <SEDRIS Abstract Base>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class either provides information about the events and/or source data used in constructing the representations specified by the scope or indicates a lack of knowledge about lineage.

Primary Page in DRM Diagram:

Secondary Pages in DRM Diagram:

Example

  1. Consider environmental data representing a transmittal containing a polygonal terrain representation that was derived from a DTED representation. The <Lineage> instance for the <Transmittal Root> instance of this example transmittal might have a <Process Step> component describing the polygonalization process as well as a <Source> component corresponding to the DTED representation.

Notes

Composed of Notes


Process_Step

If provided, each <Process Step> component of the <Lineage> instance supplies information about an event in the creation process for the representations specified by the scope.


Source

If provided, each <Source> component of the <Lineage> instance supplies information about the source data used in creating the representations specified by the scope.

Fields Notes

statement

If provided, the statement field supplies a general explanation of the data provider's knowledge about the lineage of the environmental data being described.
