Location

Abstract Class Name: Location

Superclass - <SEDRIS Abstract Base>

Subclasses

  • <Location 2D>
  • <Location 3D>
  • <Location Surface>

Definition

An instance of this DRM class specifies linear or angular quantities that designate the position of a point in an SRF.

Example

  1. A tree is instanced as a <Point Feature> instance. The <Feature Node> instance of the <Point Feature> instance shall have a <Location> component giving the position in space where the tree is placed.
