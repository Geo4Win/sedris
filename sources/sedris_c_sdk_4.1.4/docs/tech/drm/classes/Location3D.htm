Definition

An instance of this DRM class specifies linear or angular quantities that designate the position of a point in a 3D SRF.

Example

  1. A <Point Feature> instance representing a bridge is associated with a <Feature Node> instance. The location of that <Feature Node> instance in the currently scoped SRF is represented by a <Location 3D> component.

  2. The location of a <Vertex> component (point) of a <Polygon> instance in a <Model> instance representing a tank is associated with a <Geometry Node> instance. The location of that <Geometry Node> instance in the local SRF of the tank <Model> instance is represented by the <Location 3D> component of the <Vertex> instance.
