Presentation Domain

Class Name: Presentation Domain

Superclass - <SEDRIS Abstract Base>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies the set of sensor domains in which the attributed object is applicable.
