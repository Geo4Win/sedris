Quadrant Data

Class Name: Quadrant Data

Superclass - <SEDRIS Abstract Base>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies which quadrant of the environmental data is represented by the associated branch of the given quadrant related aggregation, where the environmental data as a whole corresponds to the entire aggregation.
