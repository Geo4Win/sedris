APPENDIX A - Classes
Sound Volume

Class Name: Sound Volume

Superclass - <Enclosing Volume>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies a volume within which a <Sound> instance should be active.

Primary Page in DRM Diagram:

  • 9

Secondary Pages in DRM Diagram:

  • 22

Example

  1. A cylindrical <Sound Volume> instance encompasses a river and plays the sound of running water when within that volume.
