APPENDIX A - Classes
State Data

Class Name: State Data

Superclass - <SEDRIS Abstract Base>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies the specific state value, or interval of possible state values, indicating when the associated branch of the state related aggregation is active.

The EAC specified by the state_tag of the corresponding state related aggregation provides the meaning of the field values of any given <State Data> link object.
