APPENDIX A - Classes
Volume Object

Class Name: Volume Object

Superclass - <Volume Geometry>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies a <Volume Geometry> instance using a <Location 3D> component to position the geometry and a <Volume Extent> component to define the shape.

Primary Page in DRM Diagram:

  • 5

Secondary Pages in DRM Diagram:

  • 13

Example

  1. A <Volume Object> instance represents a column of a building by specifying a cylindrical shape.

Notes

Associated with Notes


Geometry_Node

This <Geometry Node> instance corresponds to the <Location 3D> component of the <Volume Object> instance.

Composed of Notes


Union_Of_Primitive_Geometry

A <Primitive Geometry> instance P has a <Union Of Primitive Geometry> component only when P can be decomposed into nested <Primitive Geometry> instances.

EXAMPLE  A <Polygon> instance can be decomposed into multiple <Polygon> instances representing subfaces.


Location_3D

The <Location 3D> component specifies the centre of the volume specified by the <Volume Extent> component.
