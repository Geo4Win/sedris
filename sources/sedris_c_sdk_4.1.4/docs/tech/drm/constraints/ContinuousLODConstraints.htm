APPENDIX B - Constraints
Continuous LOD Constraints

Definition

The following constraints apply:

  1. <Continuous LOD Related Geometry> instances can be used only in the component tree of an <Environment Root> instance.

  2. A <Primitive Geometry> instance that is within the component tree of a <Continuous LOD Related Geometry> instance shall not have a <Union Of Primitive Geometry> component.

Rationale

  1. Nobody has flickering problems caused by zooming in on a <Model> instance.

  2. <Continuous LOD Related Geometry> instances always represent terrain, and terrain <Polygon> instances do not use subfacing.
