APPENDIX B - Constraints
Cylindrical Structure

Definition

For a <Cylindrical Volume Extent> instance C, the following conditions shall hold:

  1. The <Reference Vector> components of C shall be perpendicular to one another, and shall comply with the restrictions imposed in the specification of the <Cylindrical Volume Extent> class on their vector_type field values.

  2. If a <Reference Vector> component of C specifies a <Location> component L, L shall be the <Location 3D> instance specified by the volume context in which C appears.

Rationale

An instance of a DRM class representing a cylindrical volume shall specify a geometrically valid cylinder, and any <Location> information specified by its <Reference Vector> components shall be consistent with the context supplied by that volume.
