APPENDIX B - Constraints
Homogeneous Light Rendering Properties

Definition

A <Light Rendering Properties> instance shall contain instances of no more than one subclass of <Directional Light Behaviour>.

Rationale

This is all that is currently required and avoids clashes over which <Directional Light Behaviour> instance defines the colour of the light when viewed from a particular direction.

Example

  1. If a light has a conical lobe, it cannot also have a pyramid lobe.
