APPENDIX B - Constraints
Non Empty Environment Root Instance

Definition

An <Environment Root> instance shall have either a <Feature Hierarchy> component, a <Geometry Hierarchy> component, or both.

Rationale

An <Environment Root> instance is optional in a transmittal, but a "vacuous" <Environment Root> instance is not permitted. The DRM notation cannot enforce an "or" aggregation. Consequently, this constraint is required.
