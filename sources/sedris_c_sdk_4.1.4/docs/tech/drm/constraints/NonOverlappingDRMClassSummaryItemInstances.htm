APPENDIX B - Constraints
Non Overlapping DRM Class Summary Item Instances

Definition

In a list of <DRM Class Summary Item> instances, no two shall have matching drm_class field values.

Rationale

A <DRM Class Summary Item> indicates the presence of instances of the given class in the context being summarized. Redundancy of this information does not add value, so it is forbidden in order to simplify consumption of such lists, and to minimize the size of the summary information.
