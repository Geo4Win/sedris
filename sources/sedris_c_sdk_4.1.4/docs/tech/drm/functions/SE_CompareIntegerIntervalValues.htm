SE CompareIntegerIntervalValues
extern int
SE_CompareIntegerIntervalValues
(
const SE_Integer_Interval_Value  *a_ptr,1
const SE_Integer_Interval_Value  *b_ptr2
);

Definition

Compares two SE_Integer_Interval_Value types.

The comparison compares lower bounds, then uses upper bounds to break any ties ( see EDCS_CompareIntegerIntervals). If the lower bound and upper bound values match, closed bounds are considered less than open bounds.


Returns

-1 if a_ptr is NULL and b_ptr is non-NULL, or *a_ptr < *b_ptr
0 if both arguments are NULL, or *a_ptr == *b_ptr
1 if b_ptr is NULL and a_ptr is non-NULL, or *a_ptr > *b_ptr

Parameters Notes

1 pointer to 1st SE_Integer_Interval_Value to be compared

2 pointer to 2nd SE_Integer_Interval_Value to be compared
