SE CompareStrings
extern int
SE_CompareStrings
(
const SE_String  *a_ptr,1
const SE_String  *b_ptr2
);

Definition

Compares two SE_Strings. This function's signature is compatible with qsort.

If the locales match, this is a string comparison; otherwise, the result is determined by the locale comparison.


Returns

-1 if a_ptr is NULL and b_ptr is non-NULL, or *a_ptr < *b_ptr
0 if both arguments are NULL, or *a_ptr == *b_ptr
1 if b_ptr is NULL and a_ptr is non-NULL, or *a_ptr > *b_ptr

Parameters Notes

1 handle to 1st SE_String to be compared

2 handle to 2nd SE_String to be compared
