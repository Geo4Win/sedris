SE FreeCharacterArray
extern void
SE_FreeCharacterArray
(
char  *string_ptr,1
void  (*dealloc_func_in_ptr)(void *)2
);

Definition

Deallocates a 'string' variable. Does NOT accept an EDCS_String. Instead, this function is intended to free a (char *) that was allocated by the SE_AllocCharacterArray() function.


Parameters Notes

1 string to be freed

2 pointer to a user-supplied function to be used to free the memory. Must correspond to the function used to allocate the memory.
