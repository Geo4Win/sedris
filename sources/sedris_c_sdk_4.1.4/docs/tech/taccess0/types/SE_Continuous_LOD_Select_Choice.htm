SE_Continuous_LOD_Select_Choice 
/*
 * ENUM: SE_Continuous_LOD_Select_Choice
 *
 *   This data type specifies which level (or levels) of detail
 *   are to be traversed.
 */
typedef enum
{
SE_CLOD_ALL,1
SE_CLOD_LEAST_DETAILED,2
SE_CLOD_MOST_DETAILED3
} SE_Continuous_LOD_Select_Choice;

Enumerant Notes

1 All levels are to be traversed.

2 The least detailed level is to be traversed.

3 The most detailed level is to be traversed.
