SE_Return_Code 
/*
 * ENUM: SE_Return_Code
 *
 *   This data type specifies the function results of API functions.
 */
typedef enum
{
SE_RETCOD_FAILURE,1
SE_RETCOD_SUCCESS2
} SE_Return_Code;

Enumerant Notes

1 The function failed to carry out the operation.

2 The function successfully carried out the operation.
