SE_Sibling_Traversal_Order 
/*
 * ENUM: SE_Sibling_Traversal_Order
 *
 *   This data type specifies the desired traversal order for various
 *   types of components that already have a logical ordering.
 */
typedef enum
{
SE_SIBTRAV_ASCENDING,1
SE_SIBTRAV_DESCENDING,2
SE_SIBTRAV_UNSPECIFIED3
} SE_Sibling_Traversal_Order;

Enumerant Notes

1 Components will be traversed in ascending order.

2 Components will be traversed in descending order.

3 No traversal order is specified; any may be used.
