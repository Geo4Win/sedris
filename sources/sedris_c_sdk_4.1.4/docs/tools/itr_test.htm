SYNOPSIS

itr_test

DESCRIPTION

ITR Test uses ITR-related API calls to create three different STF transmittals named from "itr_testA.stf" through "itr_testC.stf", then creates two ITR links between them.

The transmittal "itr_testA.stf" contains a <Model Library> and a <Model> with name "AIRPLANE". Transmittal itr_testB.stf contains a <Model> named "WING" and transmittal itr_testC.stf has a <Model> named "LANDING GEAR". The <Geometry Model> component of the "AIRPLANE" <Model> contains 2 <Geometry Model Instance> objects, one of which has an ITR association relationship to the <Geometry Model> of the "WING" <Model> while the other has an ITR association relationship to the <Geometry Model> of the "LANDING GEAR" <Model>. The <Geometry Model> components of the "WING" and "LANDING GEAR" <Model>s are published. The actual geometry data in the models are meaningless. A related sedris resolution file (itr_testA.rsl) is also created in the process.

OPTIONS

None.

EXAMPLES

itr_test

Creates the 3 transmittals and their corresponding ITR resolution file as described above.
