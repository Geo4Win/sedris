SYNOPSIS

stf_test [ options ]

DESCRIPTION

STF Test makes use of the API to create a transmittal containing <Data Table> data. The program creates a transmittal named "stf_test.stf" that is populated with artificial data.

OPTIONS

number_of_components

Specifies how many components each object is to have, and how deep the object tree will be. The default value is 10.

EXAMPLES

stf_test 12

Populates a transmittal 12 levels down.
