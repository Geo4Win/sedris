EDCS Attribute Label: JAINISM_DENOMINATIONAL_FAMILY

EDCS Attribute Symbolic Constant: EAC_JAINISM_DENOMINATIONAL_FAMILY

EDCS Attribute Code: 528

Definition:
The designation of a <NON_EMPTY_SET> of closely related Jain religious denominations, often ones which were at one time part of a single movement but are now separate religious denominations; the Jainism denominational family.

EDCS Attribute Value Data Type: ENUMERATION

Code Label Symbolic constant Definition Reference type Reference Supplemental references
1 DIGAMBARA EEC_JAINDENOMFAM_DIGAMBARA Digambara. n/a    
2 SVETAMBARA EEC_JAINDENOMFAM_SVETAMBARA Svetambara. n/a    

EDCS Group Labels: RELIGION

Reference Type: IR

Reference: [WCE]

Supplemental References: none

EDCS Attribute Label: JUDAISM_DENOMINATIONAL_FAMILY

EDCS Attribute Symbolic Constant: EAC_JUDAISM_DENOMINATIONAL_FAMILY

EDCS Attribute Code: 529

Definition:
The designation of a <NON_EMPTY_SET> of closely related Jewish religious denominations, often ones which were at one time part of a single movement but are now separate religious denominations; the Judaism denominational family.

EDCS Attribute Value Data Type: ENUMERATION

Code Label Symbolic constant Definition Reference type Reference Supplemental references
1 ASHKENHAZI EEC_JUDDENOMFAM_ASHKENHAZI The largest group of Judaism, displaced from Germany, Poland, Russia, and the Ukraine; Yiddish-speaking; Ashkenhazi or "German-rite" Judaism. IR [WCE]  
2 CONSERVATIVE EEC_JUDDENOMFAM_CONSERVATIVE A movement of Judaism that began in Germany and the United States about 1900 and which opposes Reform Judaism's neglect of the Halakhah; Conservative Judaism. IR [WCE]  
3 ORIENTAL EEC_JUDDENOMFAM_ORIENTAL The third largest group of Judaism, displaced from Northern Africa and the Middle East; Arabic speaking, now mostly in Israel; Oriental Judaism. IR [WCE]  
4 ORTHODOX EEC_JUDDENOMFAM_ORTHODOX Orthodox or Chassidic Judaism. n/a    
5 REFORM EEC_JUDDENOMFAM_REFORM Liberal, Reformed, or Progressive Judaism, mostly in the United States, United Kingdom, Argentina, Brazil, Australia and New Zealand. IR [WCE]  
6 SEFARDI EEC_JUDDENOMFAM_SEFARDI The second largest division of Judaism; speaking Ladino, dating from medieval Spain but now scattered from North Africa to Afghanistan; speaking Arabic, Persian, and Aramaic; Sefardi, Shepardi, or "Spanish-rite" Judaism. IR [WCE]  
7 UNAFFILIATED EEC_JUDDENOMFAM_UNAFFILIATED Congregations not affiliated with one of the major groups of Judaism; unaffiliated. IR [WCE]

EDCS Attribute Label: JULIAN_DATE_TERRESTRIAL_TIME

EDCS Attribute Symbolic Constant: EAC_JULIAN_DATE_TERRESTRIAL_TIME

EDCS Attribute Code: 530

Definition:
The <<JULIAN_DAY>> number for the preceding noon plus the fraction of the day since that instant; the Julian date (terrestrial time). A Julian date (terrestrial time) begins at 12 hours 0 minutes 0 seconds, and the length of a day is 86 400 seconds.

EDCS Attribute Value Data Type: REAL

EDCS Unit Equivalence Class Label: PURE_NUMBER

EDCS Group Labels: TIME

Reference Type: IR

Reference: [SIED, 2.26]

Supplemental References: none

EDCS Attribute Label: JULIAN_DAY

EDCS Attribute Symbolic Constant: EAC_JULIAN_DAY

EDCS Attribute Code: 531

Definition:
The Julian day number associated with the solar day defined as the number assigned to a day in a continuous count of days beginning with the Julian day number 0 assigned to the day starting at Greenwich mean noon on 1 January 4 713 BCE, Julian proleptic calendar -4 712.

EDCS Attribute Value Data Type: COUNT

EDCS Group Labels: TIME

Reference Type: IR

Reference: [SIED, 2.26]

Supplemental References: none
