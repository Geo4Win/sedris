EDCS Attribute Label: KELP_COVERAGE

EDCS Attribute Symbolic Constant: EAC_KELP_COVERAGE

EDCS Attribute Code: 532

Definition:
A number between 0 and 1 inclusive representing the linearly-scaled fractional coverage of kelp within a delineated <WATERBODY_REGION>. Zero means no kelp is present and one means the <WATERBODY_REGION> is completely covered.

EDCS Attribute Value Data Type: REAL

EDCS Unit Equivalence Class Label: PURE_NUMBER

EDCS Group Labels: LIVING_ORGANISM, PLANT, RATE_OR_RATIO, WATERBODY_FLOOR

Reference Type: IR

Reference: [FACC, Annex B: DMK]

Supplemental References: none
