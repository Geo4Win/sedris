EDCS Classification Dictionary - Concepts whose label begin with the letter N

EDCS Classification Label: NAMED_LAND_TRACT

EDCS Classification Symbolic Constant: ECC_NAMED_LAND_TRACT

EDCS Classification Code: 788

Definition:
A <TRACT> of natural scenery, defined by its geographical characteristics and known by its proper name.

EDCS Group Labels: PHYSIOGRAPHY

Reference Type: PR

Reference: [S57A, Object: LNDRGN]

Supplemental References: none

EDCS Classification Label: NAMED_MAP_LOCATION

EDCS Classification Symbolic Constant: ECC_NAMED_MAP_LOCATION

EDCS Classification Code: 789

Definition:
A <LOCATION> that normally does not appear as a specific, characterized <OBJECT> on a <MAP> but has a name that is required to be placed on the <MAP>.

EDCS Group Labels: LOCATION, SURVEY

Reference Type: IR

Reference: [FACC, Annex A: ZD040]

Supplemental References: none

EDCS Classification Label: NATIONAL_PARK

EDCS Classification Symbolic Constant: ECC_NATIONAL_PARK

EDCS Classification Code: 790

Definition:
An extensive <REGION> of a particular nature that has been defined by law and is to be protected as a whole; it meets the prerequisites of a <NATURE_RESERVE> for the largest part and has been influenced by <HUMAN>s at most only to a small extent; a national park.

EDCS Group Labels: ANIMAL, LIVING_ORGANISM, PHYSIOGRAPHY, PLANT, RECREATION, USAGE_REGION

Reference Type: PR

Reference: [FACC, Annex A: FA080]

Supplemental References: none

EDCS Classification Label: NATURAL_AIR_OBSTRUCTION

EDCS Classification Symbolic Constant: ECC_NATURAL_AIR_OBSTRUCTION

EDCS Classification Code: 791

Definition:
An existing <OBJECT> of fixed <<HEIGHT_AGL>> or natural growth at a fixed <LOCATION> within a proscribed <REGION> (for example: adjacent to an <AERODROME>) with reference to which vertical clearance is or must be provided during flight operations; a natural air obstruction.

EDCS Group Labels: ABSTRACT_OBJECT, AIR_TRNSP, TRNSP

Reference Type: PR

Reference: [FACC, Annex A: GB220]

Supplemental References: none

EDCS Classification Label: NATURE_RESERVE

EDCS Classification Symbolic Constant: ECC_NATURE_RESERVE

EDCS Classification Code: 793

Definition:
A <REGION> that has been legally defined and the nature and landscape of which require special protection, be it in part or as a whole, in order to preserve symbioses or biotypes of specific wildlife <NON_HUMAN_ANIMAL>s or <PLANT>s: for scientific reasons; for reasons of natural or geographic history; or because of their rareness, uniqueness, or outstanding beauty; a nature reserve.

EDCS Group Labels: ANIMAL, LIVING_ORGANISM, PHYSIOGRAPHY, PLANT, RECREATION, USAGE_REGION

Reference Type: PR

Reference: [FACC, Annex A: FA081]

Supplemental References: none
