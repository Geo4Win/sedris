EDCS Classification Label: ZEBRA

EDCS Classification Symbolic Constant: ECC_ZEBRA

EDCS Classification Code: 1427

Definition:
A wild African <HORSE>, characterized by having an erect mane, whitish hide and distinctively black stripes; a zebra [SOED, "zebra", 1].

EDCS Group Labels: ANIMAL, LIVING_ORGANISM

Reference Type: n/a

Reference: none

Supplemental References: none

EDCS Classification Label: ZOO

EDCS Classification Symbolic Constant: ECC_ZOO

EDCS Classification Code: 1428

Definition:
An <ANIMAL_PARK> where wild <NON_HUMAN_ANIMAL>s are kept for study, breeding, or exhibition to the public; a zoo [SOED, "zoo", 1].

EDCS Group Labels: ANIMAL, LIVING_ORGANISM, RECREATION

Reference Type: IR

Reference: [FACC, Annex A: AK180]

Supplemental References: none
