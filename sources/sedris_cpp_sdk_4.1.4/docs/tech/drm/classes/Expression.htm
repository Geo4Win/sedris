Expression

Abstract Class Name: Expression

Superclass - <SEDRIS Abstract Base>

Subclasses

  • <Function>
  • <Literal>
  • <Variable>

Definition

An instance of this DRM class represents values that are used by <Control Link> instances and the DRM objects controlled by them.

<Expression> instances represent places within a transmittal where consuming applications shall perform an evaluation in order to get the exact value of the field of a DRM object.
