Definition

An instance of this DRM class is a spatially organized hierarchical collection of <Geometry Topology> instances, which includes all of the <Geometry Topology> instances that lie within a defined spatial extent, and are components of a single <Aggregate Geometry> instance.

FAQs

What is the purpose of this class?

This class allows <Geometry Topology> instances to be spatially organized using different methods. It ensures that all <Geometry Topology> instances, particularly those that are not directly associated with any <Geometry Representation>, have at least one containing object.

Why not restrict <Geometry Topology Hierarchy> to be a component of just <Environment Root> and <Model> - placing Topology as an organizing principle on par with <Geometry Representation> organizations and <Feature Representation> organizations?

We require the ability to spatially organize the topology of every "topological surface". A "topological surface" is not necessarily "rooted at" an <Environment Root> instance or a <Model> instance. In general, many different "topological surfaces" may exist within a <Environment Root> instance or a <Model> instance.
