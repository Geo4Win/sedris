Index LOD Data

Class Name: Index LOD Data

Superclass - <Base LOD Data>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class serves as a link object that specifies the index assigned to the associated level of detail data.

Primary Page in DRM Diagram:

  • 9

Secondary Pages in DRM Diagram:

This class appears on only one page of the DRM class diagram.

Example

  1. Consider an item represented by an <LOD Related Geometry> instance for which ten versions of the item are provided at varying levels of detail. Each version is represented by a branch of the <LOD Related Geometry> instance for which the link object is an <Index LOD Data> instance. The index values of the <Index LOD Data> link objects are 1 through 10, where number 1 corresponds to the most detailed version, and number 10 corresponds to the least detailed version.

Notes

Fields Notes

index
