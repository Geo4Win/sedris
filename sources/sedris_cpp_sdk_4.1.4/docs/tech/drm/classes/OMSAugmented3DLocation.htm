OMS Augmented 3D Location

Class Name: OMS Augmented 3D Location

Superclass - <Location 3D>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies a 3D coordinate for an SRF based on the Oblique Mercator Spherical SRFT.

See 8.5.20 Oblique Mercator spherical SRFT of ISO/IEC 18026 for a complete definition.

Primary Page in DRM Diagram:

  • 15

Secondary Pages in DRM Diagram:

This class appears on only one page of the DRM class diagram.

Example

  1. SRFs derived from the Oblique Mercator Spherical SRFT are used by the modelling and simulation community to represent METOC environmental data.
