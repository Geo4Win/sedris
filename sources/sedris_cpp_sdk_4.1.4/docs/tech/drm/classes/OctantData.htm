Class Name: Octant Data

Superclass - <SEDRIS Abstract Base>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies which octant of the entire aggregation that the associated branch of the given octant related aggregation represents.
