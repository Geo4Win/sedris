Perimeter Related Geometry Topology

Class Name: Perimeter Related Geometry Topology

Superclass - <Geometry Topology Hierarchy>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class is a <Geometry Topology Hierarchy> instance that spatially organizes its components into a collection of (potentially) irregularly shaped regions, each defined by a <Perimeter Data> link object.

Primary Page in DRM Diagram:

  • 11

Secondary Pages in DRM Diagram:

This class appears on only one page of the DRM class diagram.

Example

  1. Consider a plat of a residential subdivision whose streets and driveways are represented using <Polygon> instances organized by a <Perimeter Related Geometry> instance. To represent the geometric topology of the subdivision, the <Perimeter Related Geometry> instance has a <Perimeter Related Geometry Topology> component that represents the set of all <Geometry Topology> instances showing how the driveways of houses in the subdivision are connected to the streets of the subdivision.

geometry_topology_level
