APPENDIX A - Classes
Sound Library

Class Name: Sound Library

Superclass - <Library>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies the complete list of the unique sounds whose references are resident within a transmittal.

Primary Page in DRM Diagram:

  • 10

Secondary Pages in DRM Diagram:

  • 1
  • 22

Example

  1. A transmittal represents environmental data representing a littoral region. The different sounds that can occur within the littoral region are referenced from the <Sound Library> instance.
