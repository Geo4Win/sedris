APPENDIX A - Classes
Spatial Association Data

Class Name: Spatial Association Data

Superclass - <Base Spatial Association Data>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class is a link object that specifies the semantic meaning of the association relationship with which it is bound. This semantic meaning specifies a spatial relationship.

Primary Page in DRM Diagram:

  • 9

Secondary Pages in DRM Diagram:

This class appears on only one page of the DRM class diagram.

Example

  1. Consider two <Linear Feature> instances, one representing a river and the other representing a bridge crossing the river. A <Spatial Association Data> link object on an association from the bridge to the river has meaning field set to SE_SPATASSMEANTYP_COMPLETELY_ABOVE.
