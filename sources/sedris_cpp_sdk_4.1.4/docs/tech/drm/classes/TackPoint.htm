Tack Point

Class Name: Tack Point

Superclass - <SEDRIS Abstract Base>

Subclasses

This DRM class is concrete and has no subclasses.

Definition

An instance of this DRM class specifies a <Location> instance that corresponds to a fixed position on one or more images depending on the number of corresponding <Image Mapping Function> instances that are components of the aggregating <Primitive Geometry> instance.

Primary Page in DRM Diagram:

  • 5

Secondary Pages in DRM Diagram:

This class appears on only one page of the DRM class diagram.

Example

  1. Consider a <Polygon> instance P for which the data provider wishes to supply a texture coordinate at the centre of P rather than at one of its vertices To achieve this, the data provider supplies a <Tack Point> component K for P such that K specifies the centre of P as its <Location> component and the desired texture coordinate as a <Texture Coordinate> component.
