Definition

An instance of DRM class specifies an interval of time, defined by a start time and a stop time.

FAQs

What is the purpose of this class?

This class provides 1) ISO 19115-compliant metadata that describes the time period within which a high-level DRM object (e.g., an instance of <Transmittal Root>, <Model>, or <Image>) is valid, and 2) a general-purpose mechanism for describing time intervals. The <Time Interval> allows potential users of a transmittal to evaluate the time period covered by the transmittal, without necessarily having to actually obtain or examine the transmittal itself.
