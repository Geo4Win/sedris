Transformation

Abstract Class Name: Transformation

Superclass - <SEDRIS Abstract Base>

Subclasses

  • <LSR Transformation>
  • <World Transformation>

Definition

An instance of this DRM class is applied to the DRM object aggregating the instance. <Location 3D> instances and <Reference Vector> instances within the scope of the aggregating DRM object are always affected by <Transformation> instances.
