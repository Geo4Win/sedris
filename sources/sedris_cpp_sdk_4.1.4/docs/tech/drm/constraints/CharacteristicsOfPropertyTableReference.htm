APPENDIX B - Constraints
Characteristics Of Property Table Reference

Definition

For any <Property Table Reference> instance R specifying a <Property Table> instance T:

  1. T shall be a component of a <Data Table Library> instance.

  2. T shall have an <Axis> component, the axis_type of which corresponds to that specified by R.

  3. The <Axis> instance thus specified shall have axis_value_count greater than or equal to (index_on_axis + 1), where index_on_axis is that specified by R.

Rationale

A <Property Table Reference> instance specifies an N-1 dimensional slice of an N dimensional <Property Table> component of a <Data Table Library> instance. Consequently, the <Axis> instance and tick mark value specified by the <Property Table Reference> instance shall correspond to part of the <Property Table> instance being referenced.
