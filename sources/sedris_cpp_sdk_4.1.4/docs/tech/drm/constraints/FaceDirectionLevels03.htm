APPENDIX B - Constraints
Face Direction Levels 0 3

Definition

At feature topology levels 0 though 3, the front field of a <Face Direction> instance shall always be SE_TRUE.

Rationale

Only when feature topology is 3-dimensional can a <Feature Face> instance be said to have both a front and a back side.
