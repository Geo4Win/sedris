APPENDIX B - Constraints
Inheritance Rule For Location

Definition

Given any DRM object that has a <Location> component, that <Location> component (or the first <Location> component in an ordered list of <Location> components) becomes the (default) <Location> component in the context for the component tree stemming from that DRM object.
