SE CompareURNs
extern int
SE_CompareURNs
(
const SE_URN  *a_ptr,1
const SE_URN  *b_ptr2
);

Definition

Compares two SE_URNs. This function's signature is compatible with qsort.


Returns

-1 if a_ptr is NULL and b_ptr is non-NULL, or *a_ptr < *b_ptr
0 if both arguments are NULL, or *a_ptr == *b_ptr
1 if b_ptr is NULL and a_ptr is non-NULL, or *a_ptr > *b_ptr

Parameters Notes

1 handle to 1st SE_URN to be compared

2 handle to 2nd SE_URN to be compared
