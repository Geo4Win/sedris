SE EnumNameToAPI
extern SE_DRM_Status_Code
SE_EnumNameToAPI
(
const char  *enum_name_in_ptr,1
SE_API  *result_out_ptr2
);

Definition

Retrieves the SE_API corresponding to the given string.


Returns

SE_DRM_STAT_CODE_SUCCESS and *result_out_ptr is set appropriately, if valid parameters were passed in and all operations succeeded.
SE_DRM_STAT_CODE_NULL_REQUIRED_PARAMETER and no changes are made, if any parameter was NULL.
SE_DRM_STAT_CODE_FAILURE and no changes are made, if enum_name_in_ptr doesn't correspond to an enumerant.

Parameters Notes

1 enumerant name to be processed

2 pointer to a variable in the user's memory space where the result will be stored.
