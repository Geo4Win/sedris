Definition

Identifies the superclass of the given DRM class, if any.


Returns

tag of drm_class' superclass if a valid parameter was passed in.
SE_CLS_DRM_NULL if drm_class is not valid, or is valid but has no superclass.

Parameters Notes

1 represents a DRM class.
