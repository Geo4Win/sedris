SE ValidNonNullString
extern SE_Boolean
SE_ValidNonNullString
(
const SE_String  *value_ptr,1
SE_Boolean  print_error,2
SE_Integer_Unsigned  *error_count_ptr3
);

Definition

Checks that the given SE_String is valid and contains a non-empty string.


Returns

SE_TRUE if value_ptr is non-NULL and *value_ptr contains valid information
SE_FALSE otherwise

Parameters Notes

1 pointer to SE_String, the contents of which are to be checked

2 whether to print error messages

3 pointer to a variable in which the error count for this SE_String is stored; if NULL, no counting is performed
