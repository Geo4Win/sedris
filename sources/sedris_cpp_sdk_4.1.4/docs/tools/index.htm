Introduction

The SEDRIS SDK includes several applications that can be used to verify transmittals and examine their contents. They can also be used as a starting point for developing other SEDRIS applications.

For general information about this release, where to obtain it, and items that require specific attention, see the Release Notes.

For help, comments, and bug reports please send email to help@sedris.org. If you are an associate, please use se-coders@sedris.org.

Tools

The following tools are included in the SEDRIS SDK:

Depth

An application that traverses an STF transmittal and displays the objects in text format. See a sample run of the Depth tool.

Model Viewer

A 3D viewer that can display <Model> geometry and textures using OpenGL. See a sample run of the Model Viewer tool.

Rules Checker (C SDK only)

A tool that checks the semantical correctness of an STF transmittal. See a sample run of the Rules Checker tool.

Syntax Checker (C SDK only)

A tool that checks the syntactical correctness of an STF transmittal. See a sample run of the Syntax Checker tool.

Utilities

The following utilities are included in the SEDRIS SDK:

ITR Test

An application that demonstrates the use of the Inter-Transmittal Referencing mechanism to link multiple STF transmittals.

STF Test

A sample application that creates and populates an STF transmittal with <Data Table> data.

Other Tools

There are several other SEDRIS tools available not included in the SDK. You can find them at the SEDRIS Tools Web Site. Some of these tools are:

Converters

Several applications that can convert from other formats (like CTDB, VPF, DTED, GDF) to/from STF.

SEE-IT

An inspection and analysis tool that can find problems with Synthetic Natural Environments and helps to correct them.

Side by Side Viewer

An application that can display the hierarchical structure of a transmittal and visualize the 3D geometry of models and terrain.

STF Util

A utility program that can query and perform maintenance activities on STF transmittals, such as renaming the transmittal, removing unused space, displaying ITR references and resolve URNs, and displaying specific file/block/object data.

SEDRIS Transmittal Browser

An application that displays the hierarchical structure of a transmittal with a point-and-click interface.
