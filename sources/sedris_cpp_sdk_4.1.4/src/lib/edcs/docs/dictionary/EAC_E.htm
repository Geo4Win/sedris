EDCS Attribute Dictionary - Concepts whose labels begin with the letter E

EDCS Attribute Label: ECOSYSTEM_TYPE

EDCS Attribute Symbolic Constant: EAC_ECOSYSTEM_TYPE

EDCS Attribute Code: 321

Definition:
The type of an ecosystem based on a combination of land cover, floristic properties, climate, and physiognomy.

EDCS Attribute Value Data Type: ENUMERATION
