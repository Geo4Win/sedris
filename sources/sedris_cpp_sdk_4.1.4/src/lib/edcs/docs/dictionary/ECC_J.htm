EDCS Classification Dictionary - Concepts whose label begin with the letter J

EDCS Classification Label: JAIL

EDCS Classification Symbolic Constant: ECC_JAIL

EDCS Classification Code: 597

Definition:
A local <DETENTION_FACILITY> where the criminally accused are initially taken to be processed and await trial and where short (usually less than one year) sentences for criminal misconduct can be served; a jail.

EDCS Group Labels: ADMINISTRATION, USAGE_REGION

Reference Type: n/a

Reference: none

Supplemental References: [SOED, "jail", A1]

EDCS Classification Label: JETTY

EDCS Classification Symbolic Constant: ECC_JETTY

EDCS Classification Code: 598

Definition:
A man-made <BARRIER> built out into or in a <WATERBODY>, primarily to restrain or direct <WATER_CURRENT>s and <WATER_WAVE>s; a jetty.

EDCS Group Labels: HYDROGRAPHIC_ARTEFACT, WATERBODY_SURFACE

Reference Type: PR

Reference: [FACC, Annex A: BB140]

Supplemental References: none

EDCS Classification Label: JUNGLE

EDCS Classification Symbolic Constant: ECC_JUNGLE

EDCS Classification Code: 599

Definition:
A <TRACT>, especially tropical, overgrown with luxuriant and often almost impenetrable <VEGETATION>; a jungle [SOED, "jungle", 1].

EDCS Group Labels: LIVING_ORGANISM, PHYSIOGRAPHY, PLANT

Reference Type: n/a

Reference: none

Supplemental References: none

EDCS Classification Label: JUPITER

EDCS Classification Symbolic Constant: ECC_JUPITER

EDCS Classification Code: 600

Definition:
The primary <PLANET> of the <SUN> that is fifth in distance from the <SUN>, with an orbital mean <<RADIUS>> of approximately 778,30 million kilometres; Jupiter.

EDCS Group Labels: SPACE

Reference Type: IR

Reference: [PSMOP]

Supplemental References: none
