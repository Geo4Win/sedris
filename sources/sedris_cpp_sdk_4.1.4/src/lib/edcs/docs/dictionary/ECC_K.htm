EDCS Classification Label: KELP_BED

EDCS Classification Symbolic Constant: ECC_KELP_BED

EDCS Classification Code: 601

Definition:
A <WATERBODY_FLOOR_REGION> covered with a significant concentration of kelp; a kelp bed.

EDCS Group Labels: LIVING_ORGANISM, PHYSIOGRAPHY, PLANT, WATERBODY_FLOOR

Reference Type: n/a

Reference: none

Supplemental References: none

EDCS Group Labels: ANIMAL, LIVING_ORGANISM

EDCS Classification Label: KILN

EDCS Classification Symbolic Constant: ECC_KILN

EDCS Classification Code: 603

Definition:
An oven for hardening, burning, or drying <MATERIAL>s (for example: grain, meal, and/or clay), especially a brick-lined oven used to bake or fire ceramics; a kiln [AHD, "kiln"].

EDCS Group Labels: INDUSTRY

Reference Type: n/a

Reference: none

Supplemental References: none

EDCS Classification Label: KUIPER_BELT

EDCS Classification Symbolic Constant: ECC_KUIPER_BELT

EDCS Classification Code: 604

Definition:
The <REGION> of the <SOLAR_SYSTEM> of the <SUN> beyond the orbit of <NEPTUNE> where a large number of <COMET>s with short orbital periods originate; the Kuiper belt.

EDCS Group Labels: SPACE

Reference Type: IR

Reference: [BSFG]
