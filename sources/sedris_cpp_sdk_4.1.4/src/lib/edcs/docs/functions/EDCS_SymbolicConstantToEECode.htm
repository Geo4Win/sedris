EDCS SymbolicConstantToEECode
extern EDCS_Status_Code
EDCS_SymbolicConstantToEECode
(
const EDCS_Character  *ee_symbolic_constant_in_ptr,1
EDCS_Attribute_Code  *ea_out_ptr,2
EDCS_Enumerant_Code  *ee_out_ptr3
);

Definition

Retrieves the {EA, EE} code pair corresponding to the given symbolic constant.


Returns

EDCS_SC_SUCCESS and *ea_out_ptr, *ee_out_ptr are set to the appropriate values, if valid parameters were passed in.
EDCS_SC_NULL_REQUIRED_PARAMETER and no changes are made, if any parameter is NULL.
EDCS_SC_INVALID_INPUT_SYMBOLIC_CONSTANT and no changes are made, if *ee_symbolic_constant_in_ptr does not correspond to a valid EE for some EA.

Parameters Notes

1 the symbolic constant for which the corresponding EE code is requested; note that this is the language-binding symbolic constant name, rather than the abstract label (i.e., corresponds to the C language binding standard rather than the abstract EDCS specification)

2 pointer to a variable in the user's memory into which the output EAC will be copied

3 pointer to a variable in the user's memory into which the output EEC will be copied
