EDCS Map3p1AttributeCode
extern EDCS_Status_Code
EDCS_Map3p1AttributeCode
(
EDCS_Integer  ea_3p1_code_in,1
const EDCS_3p1_4px_Attribute_Mapping_Structure  **mapping_info_out_ptr2
);

Definition

Retrieves forward mapping information for the given code representing an EDCS 3.1 attribute code.


Returns

EDCS_SC_SUCCESS and a handle to the requested mapping information is copied into *mapping_info_out_ptr, if valid parameters were passed in.
EDCS_SC_NULL_REQUIRED_PARAMETER and *mapping_info_out_ptr is left unaltered, if mapping_info_out_ptr was NULL.
EDCS_SC_INVALID_INPUT_CODE and *mapping_info_out_ptr is set to NULL, if *ea_3p1_code_in does not correspond to a valid EDCS 3.1 EA.

Parameters Notes

1 the EDCS 3.1 attribute code whose mapping information is being retrieved.

2 pointer to a pointer variable in the user's memory space where a handle to the mapping information will be copied.
