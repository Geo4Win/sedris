EDCS Attribute Label: NAVIGATION_LIGHT_INTENSITY

EDCS Attribute Symbolic Constant: EAC_NAVIGATION_LIGHT_INTENSITY

EDCS Attribute Code: 872

Definition:
A number between 0 and 1 inclusive representing the linearly-scaled fractional intensity of external <LIGHTING> that is used to illuminate the environment in the <DIRECTION> of travel of a <VEHICLE>; the navigation or head light intensity. Zero means minimum intensity and one means maximum intensity.
