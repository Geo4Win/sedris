EDCS Attribute Label: TANKER_DENSITY

EDCS Attribute Symbolic Constant: EAC_TANKER_DENSITY

EDCS Attribute Code: 1402

Definition:
The areal density of tanker <WATER_SURFACE_VESSEL>s.
