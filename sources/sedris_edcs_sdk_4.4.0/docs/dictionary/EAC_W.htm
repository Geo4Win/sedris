EDCS Attribute Label: WAKE_INTENSITY

EDCS Attribute Symbolic Constant: EAC_WAKE_INTENSITY

EDCS Attribute Code: 1546

Definition:
A number between 0 and 1 inclusive representing the linearly-scaled fraction of the production rate of <WATERBODY_SURFACE> wakes induced by the movement of a <VESSEL> relative to <WATER_WAVE>s and <WATER_CURRENT>s. Zero means no wake at all and one means maximum intensity.

EDCS Attribute Value Data Type: REAL

EDCS Unit Equivalence Class Label: PURE_NUMBER

EDCS Group Labels: HYDROGRAPHIC_TRNSP, RATE_OR_RATIO, TRNSP, WATERBODY_SURFACE

Reference Type: IR

Reference: [EBEV, 4.3.1.3, Bits: 7-8 (Trailing Effects)]

Supplemental References: none
