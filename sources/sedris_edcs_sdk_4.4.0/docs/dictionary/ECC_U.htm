EDCS Classification Dictionary - Concepts whose label begin with the letter U

EDCS Classification Label: UNCONTROLLED_AIRSPACE

EDCS Classification Symbolic Constant: ECC_UNCONTROLLED_AIRSPACE

EDCS Classification Code: 1297

Definition:
Any <AIRSPACE> in which air traffic control service is not available; an uncontrolled airspace.

EDCS Group Labels: AIR_TRNSP, TRNSP

Reference Type: IR

Reference: [AIM]

Supplemental References: none
