EDCS Unit Equivalence Class Label: FIELD_OR_POWER_LEVEL_DIFF

Unit Status: Accepted

Number of quantities which this unit of measure may represent:: 4

  • Quantity: field quantity ratio (level difference)
    Definition: Twice the common logarithm of a field quantity ratio.
    1 B is the level of a field quantity when 2 lg (F/F0) = 1 where F and F0 represent two amplitudes of the same kind, F0 being a reference amplitude.
    1 B = (1/2) ln 10 Np = 1,151 293 Np (approximately).
    Reference type: PR
    Reference: [I31-2, 2-9.b]
    Supplemental References: none

  • Quantity: power quantity ratio (level difference)
    Definition: The common logarithm of a power quantity ratio.
    1 B is the level of a power quantity when lg (P/P0) = 1 where P and P0 represent two powers, P0 being a reference power.
    Reference type: PR
    Reference: [I31-2, 2-10.b]
    Supplemental References: none

  • Quantity: sound power level
    Definition: LW = 1/2 ln (P/P0) = 1/2 ln 10·lg (P/P0) where P is the root-mean-square of the sound power and the reference power p0 = 1 x 10-12 W.
    Reference type: PR
    Reference: [I31-7, 7-22.a]
    Supplemental References: none

  • Quantity: sound pressure level
    Definition: Lp = ln (p/p0) = ln 10·lg (p/p0) where p is the root-mean-square value of the sound pressure and the reference pressure p0 = 20 μPa.
    Reference type: PR
    Reference: [I31-7, 7-21.a]
    Supplemental References: none
