Defects Fixed Since Release 4.0.0 Through Release 4.3.1

  • Most changes made in compliance with the following SEDRIS ISO FDIS specifications:
    • Environmental Data Encoding Specification (EDCS) (ISO/IEC 18025)
    • Environmental Data Encoding Specification C Language Binding (ISO/IEC 18041-4)
  • Fixed sorting problem of country codes in EDCS_Locale validation.
  • Missing edcs3p1_to_edcs4p0.h include file
  • Corrected mappings from EDCS 3.1 to EDCS 4.0
  • Improved attribute value validation.
