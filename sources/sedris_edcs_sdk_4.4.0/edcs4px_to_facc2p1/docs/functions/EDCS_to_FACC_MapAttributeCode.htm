Concept Mapping from EDCS 4.4 to FACC 2.1
Functions
EDCS to FACC MapAttributeCode
extern EDCS_Status_Code
EDCS_to_FACC_MapAttributeCode
(
EDCS_Attribute_Code ea_code_in, (notes)
FACC_Attribute_Mapping_Structure **mapping_info_out_ptr (notes)
);

Definition

Retrieves FACC 2.1 mapping information for the given EDCS Attribute Code.


Returns

EDCS_SC_SUCCESS and a handle to the requested mapping information is copied into *mapping_info_out_ptr, if valid parameters were passed in.
EDCS_SC_NULL_REQUIRED_PARAMETER and *mapping_info_out_ptr is left unaltered, if mapping_info_out_ptr was NULL.
EDCS_SC_OTHER_FAILURE and *mapping_info_out_ptr is left unaltered, if the mapping library has not been initialized with EDCS_to_FACC_InitMappingInfo().
EDCS_SC_INVALID_CODE and *mapping_info_out_ptr is set to NULL, if ea_code_in does not correspond to a valid EDCS 4.4 Attribute.

Parameters Notes

ea_code_in

the EDCS 4.4 Attribute code whose mapping information is being retrieved

mapping_info_out_ptr

pointer to a pointer variable in the user's memory space where a handle to the mapping information will be copied
