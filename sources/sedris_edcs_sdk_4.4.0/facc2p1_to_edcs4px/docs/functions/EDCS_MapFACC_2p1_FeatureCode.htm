Functions
EDCS MapFACC_2p1_FeatureCode
extern EDCS_Status_Code
EDCS_MapFACC_2p1_FeatureCode
(
constFACC_2p1_Feature_Code *fac_code_in_ptr, (notes)
constFACC_2p1_Class_Mapping_Structure **mapping_info_out_ptr (notes)
);

Definition

Retrieves mapping information for the given FACC 2.1 Feature Code.

Returns

EDCS_SC_SUCCESS and a handle to the requested mapping information is copied into *mapping_info_out_ptr, if valid parameters were passed in.
EDCS_SC_NULL_REQUIRED_PARAMETER and *mapping_info_out_ptr is left unaltered, if any parameter was NULL.
EDCS_SC_INVALID_CODE and *mapping_info_out_ptr is set to NULL, if *fac_code_in_ptr does not correspond to a valid EDCS 4.4 Classification.

Parameters Notes

fac_code_in_ptr

the FACC 2.1 Feature Code whose mapping information is being retrieved

mapping_info_out_ptr

pointer to a pointer variable in the user's memory space where a handle to the mapping information will be copied
